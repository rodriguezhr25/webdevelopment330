Arrays

Removing Values from Arrays

The delete operator will remove an item from an array:
delete avengers[3]; //true

Methods

  • To remove the last item from an array, we can use the pop() method.
  • The shift() method works in a similar way to the pop() method, but this removes the first item in the array.
  • The push() method appends a new value to the end of the array.
  • The unshift() method is similar to the push() method, but this appends a new item to the beginning of the array.
  • The concat() method can be used to merge an array with one or more arrays.
  • The join() method can be used to turn the array into a string that comprises all the items in the array, separated by commas.
  • The slice() method creates a subarray; effectively chopping out a slice of an original array, starting at one position and finishing at another.
    avengers.slice(2,4) // starts at the third item (index of 2) and finishes at the fourth (the item with index 4 is not included)
  • The splice() method removes items from an array then inserts new items in their place.
    For example, the following code removes the string 'Hulk' and replaces it with 'Scarlett Witch':
    avengers['Captain America', 'Iron Man', 'Thor', 'Hulk', 'Hawkeye', 'Black Widow']
    avengers.splice(3, 1, 'Scarlet Witch');
  • We can reverse the order of an array using the reverse() method.
  • We can sort the order of an array using the sort() method. Not use for numbers.
  • Find the index of one item.
    vengers.indexOf('Iron Man');
  • ES6 also introduced the includes() method. This returns a boolean value depending on whether the array contains a particular element or not:
    avengers.includes('Iron Man');
    // true
    avengers.includes('Thor');
    //false

Sets

A set is a data structure that represents a collection of unique values, so it cannot include any duplicate values.
const list = new Set();
list.add(1);
//If you try to add a value that is already contained in the set, then the operation is simply ignored.
const numbers = new Set([1,2,3]);
The number of values in a set can be found using the size() method.
numbers .size()
//3 The has() method can be used to check if a value is in a set. This returns a boolean value of true or false
numbers.has(1);
// true
numbers.has(4);
//false
Sets do not have index notation for inspecting individual entries. It cannot be iterated as an array using an index.
The delete() method allows eliminating one specific item of the set and the clear() method removes all values.

Converting set to an array.

const shoppingSet = new Set().add('Apples').add('Bananas').add('Beans');
shoppingSet
//Set { 'Apples', 'Bananas', 'Beans' }
A set can be converted into an array by placing the set, along with the spread operator (...) directly inside an array literal.
const shoppingArray = [...shoppingSet]
shoppingArray
// [ 'Apples', 'Bananas', 'Beans' ]
It's also possible to use the Array.from() method to convert a set into an array.
const shoppingArray = Array.from(shoppingSet);

Creating Maps

They are a convenient way of keeping a list of key and value pairs, and are similar to 'hashes', or 'hash tables' or 'dictionaries' in other programming languages.
const romanNumerals = new Map();
The set() method can be used to add a key and value pair to a map.
romanNumerals.set(2,'II').set(3,'III').set(4,'IV').set(5,'V');
// Map { 1 => 'I', 2 => 'II', 3 => 'III', 4 => 'IV', 5 => 'V' }
To look up a value, we can use the get() method
romanNumerals.get(4);
// 'IV'
The has() method can be used to check if a particular key is in a map. This returns a boolean value of true or false :
romanNumerals.has(5);
The delete() method can be used to remove a key and value pair from a map, and the clear() method removes all key and value pairs.
const heroes = new Map([ ['Clark Kent','Superman'],
['Bruce Wayne', 'Batman']
]);
heroes.size
//2
heroes.delete('Clark Kent');
// true
heroes.size
// 1
